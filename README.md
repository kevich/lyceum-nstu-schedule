## Lyceum NSTU Schedule

[![Go build status](https://github.com/kevich/lyceum-nstu-schedule/actions/workflows/go.yml/badge.svg?branch=main)](https://github.com/kevich/lyceum-nstu-schedule/actions/workflows/go.yml)
[![Coverage Status](https://coveralls.io/repos/github/kevich/lyceum-nstu-schedule/badge.svg?branch=main)](https://coveralls.io/github/kevich/lyceum-nstu-schedule?branch=main)

A Go application that scrapes and transforms NSTU Lyceum schedule data into a structured format suitable for Yandex.Alice skill integration.

### Features
- Converts NIKA JSON to human-readable schedule structures
- Handles date ranges, replacements, and cancellations
- Minimal dependencies; pure transformation logic
- Ready for Yandex.Alice voice assistant integration

### Development
Prerequisites: Go 1.21+
